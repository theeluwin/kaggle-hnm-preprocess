# Kaggle H&M Preprocess

IDS 연구실의 2022년도 여름방학~2학기 UROP 과목 참고용 자료.

[Kaggle H&M Challange](https://www.kaggle.com/competitions/h-and-m-personalized-fashion-recommendations) 참고.

일반적인 평가라던가 데이터 전처리라던가 모두 담당.

anaconda3 환경을 가정함.
